Available Courses

Introduction to Programming

An introductory course that teaches the fundamentals of programming using a high-level programming language. Topics covered include variables, data types, control structures, functions, and problem-solving techniques.

Data Structures and Algorithms

This course focuses on the study of fundamental data structures such as arrays, linked lists, stacks, queues, trees, and graphs. It also covers various algorithms and their analysis, including sorting, searching, and graph algorithms.

Database Systems

An introduction to database systems covering concepts of data modeling, relational databases, SQL queries, normalization, and transaction management. Students will gain hands-on experience with popular database management systems.

Operating Systems

This course explores the design and implementation of operating systems. Topics include process management, memory management, file systems, and device management. Students will gain practical experience through programming assignments.

Web Development

A course that covers the fundamental technologies and techniques for web development, including HTML, CSS, JavaScript, and server-side programming. Students will learn how to create interactive and dynamic web applications.

Artificial Intelligence

This course covers the fundamentals of artificial intelligence, including search algorithms, knowledge representation, reasoning, machine learning, and natural language processing. Students will work on AI projects.
